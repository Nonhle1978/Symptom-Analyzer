Symptom Analyzer

The Symptom Analyzer is a simple web-based prototype that helps users assess the urgency of their symptoms before visiting a clinic.
This tool does not provide medical diagnosis but provides guidance based on common symptom keywords.

Features:
-Text input for describing symptoms
-Analyze button to classify urgency level
-Displays Low, Medium, or High urgency
-Color-coded results for easy understanding
-Simple, responsive, beginner-friendly UI
-Clear disclaimer about non-diagnosis

Usage:
1. Enter your symptoms in the textarea (e.g., “I have chest pain and dizziness”).
2. Click Analyze Symptoms.
3. The result card will display:
  -Urgency Level (Low / Medium / High)
  -Recommendation message

Disclaimer:
  This tool does not provide medical diagnosis.
  It is intended as a prototype and educational project only.
  Always consult a healthcare professional for serious symptoms.
